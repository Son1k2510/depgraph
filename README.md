# Dependency Graph Analyzer

## Описание проекта

Проект представляет собой инструмент визуализации графа зависимостей для пакетов на языке Rust (Cargo).  
Цель проекта — получить граф зависимостей пакета, провести его анализ и визуализировать в виде графа и ASCII-дерева.

Проект реализован поэтапно, с возможностью настройки через конфигурационный файл формата INI.

---

## Функциональность

### Этап 1: Минимальный прототип с конфигурацией
- CLI-приложение с настройками через конфигурационный файл `config.ini`.
- Настраиваемые параметры:
  - `package_name` — имя анализируемого пакета.
  - `repo_url` — URL репозитория или путь к тестовому репозиторию.
  - `test_mode` — режим работы с тестовым репозиторием.
  - `output_file` — имя файла для сохранения изображения графа.
  - `ascii_tree` — режим вывода зависимостей в виде ASCII-дерева.
  - `max_depth` — максимальная глубина анализа зависимостей.
- При запуске приложения выводятся все параметры в формате ключ-значение.
- Реализована проверка и обработка ошибок для всех параметров.

### Этап 2: Сбор данных
- Получение прямых зависимостей заданного пакета из репозитория Cargo.
- Вывод на экран всех прямых зависимостей.
- Реализован доступ к данным без использования менеджеров пакетов и сторонних библиотек для анализа зависимостей.

### Этап 3: Основные операции над графом
- Построение графа зависимостей с учетом транзитивных зависимостей.
- Реализован алгоритм DFS с рекурсией.
- Поддержка максимальной глубины анализа.
- Обработка циклических зависимостей.
- Поддержка тестового режима работы с файлами описания графа зависимостей.
- Демонстрация работы на различных тестовых репозиториях.

### Этап 4: Дополнительные операции
- Вывод порядка загрузки зависимостей для заданного пакета.
- Сравнение результатов с реальным менеджером пакетов.
- Демонстрация работы на различных тестовых репозиториях.
